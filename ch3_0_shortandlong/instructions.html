実験説明

プレイヤーは「顧客」と「企業」です。

顧客の戦略は「仕事を発注しない」「仕事を発注する」です。

企業の戦略は「仕事をする」「仕事をしない」です。

顧客が先手で、企業が後手の展開形ゲームだとします。顧客が「仕事を発注しない」ならば、お互いに０ポイントです。

顧客が「仕事を発注する」を選び、企業が「仕事をする」を選ぶと、顧客に２ポイント、企業に２ポイントです。

顧客が「仕事を発注する」を選び、企業が「仕事をしない」を選ぶと、顧客に-1ポイント、企業に4ポイントです。

{% if C.NUM_ROUNDS == subsession.round_number %}ただし、「もし仕事をしない」を選択した場合はこの顧客から「二度と仕事がもらえない」とします。{{ endif }}

※入力がなければ、コンピュータによりランダムに決定されます。
